Simple Audio Annotator

Quickstart

  1. Load file
  2. Press "space bar" or "play" to start listening
  3. Press "alt + a" to add start timestamp
  4. Press "alt + e" to add end timestamp to first unset timtestamp group.
  5. Press "alt + t" to add text to first unset timestamp group
  6. Press "Delete" button to delete each timestamp group
  7. Press "Download" button to download the content of the timestamps annotations in .txt format

All time stamp fields are editable if you click once on them.

Limitations

  • Download retrieves everything in the Timestamps block, in .txt format.
  • Only way to change file loaded is refreshing the page.
  • No local storage, if you refresh the page you lose your progress. Download content often.
